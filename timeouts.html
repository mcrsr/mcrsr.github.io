TimeOuts

An interface for managing timeout behavior for WebDriver instances.

Method Description
implicitlyWait(long time, java.util.concurrent.TimeUnit unit) Specifies the amount of time the driver should wait when searching for an element if it is not immediately present.
pageLoadTimeout(long time, java.util.concurrent.TimeUnit unit) Sets the amount of time to wait for a page load to complete before throwing an error.
setScriptTimeout(long time, java.util.concurrent.TimeUnit unit) Sets the amount of time to wait for an asynchronous script to finish execution before throwing an error.

implicitlyWait WebDriver.Timeouts implicitlyWait(long time, java.util.concurrent.TimeUnit unit) Specifies the amount of time the driver should wait when searching for an element if it is not immediately present. When searching for a single element, the driver should poll the page until the element has been found, or this timeout expires before throwing a NoSuchElementException. When searching for multiple elements, the driver should poll the page until at least one element has been found or this timeout has expired. Increasing the implicit wait timeout should be used judiciously as it will have an adverse effect on test run time, especially when used with slower location strategies like XPath. 

                            
    import java.util.concurrent.TimeUnit;
    import org.openqa.selenium.WebDriver;
    import org.openqa.selenium.chrome.ChromeDriver;
    import io.github.bonigarcia.wdm.WebDriverManager;
      
    public class TimeOutsExample1 {
      public static void main(String[] args) throws Exception {
        WebDriverManager.chromedriver().setup();
        WebDriver driver = new ChromeDriver();
        driver.manage().window().maximize();
        driver.manage().timeouts().implicitlyWait(10, TimeUnit.SECONDS);
        driver.manage().timeouts().pageLoadTimeout(10, TimeUnit.SECONDS);
        driver.manage().timeouts().setScriptTimeout(10, TimeUnit.SECONDS);
        driver.get("https://www.w3schools.com");
        driver.quit();
      }
    }


Scripttimeout example:

                            
    import java.util.concurrent.TimeUnit;
    import org.openqa.selenium.JavascriptExecutor;
    import org.openqa.selenium.WebDriver;
    import org.openqa.selenium.chrome.ChromeDriver;
    import io.github.bonigarcia.wdm.WebDriverManager;
      
    public class ScriptTimeoutExample1 {
      public static void main(String[] args) throws Exception {
        WebDriver driver = null;
        try {
          WebDriverManager.chromedriver().setup();
          driver = new ChromeDriver();
          driver.manage().window().maximize();
          driver.manage().timeouts().setScriptTimeout(5, TimeUnit.SECONDS);
          driver.get("https://www.w3schools.com");
          JavascriptExecutor jse = ((JavascriptExecutor)driver);
          jse.executeAsyncScript("setTimeout(()=>{alert('Hello from alert');},6000)");
        }catch (Exception e) {
          e.printStackTrace();
        }
        driver.quit();
      }
    }


If you run the above program you will get following error:

org.openqa.selenium.ScriptTimeoutException: script timeout
