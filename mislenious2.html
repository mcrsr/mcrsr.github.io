Scrolling the page using selenium Java

Posted by MCRSR on Sept 04, 2023

How can we scroll the page using selenium java

To scroll a web page using Selenium in Java, we can use the JavascriptExecutor interface to execute JavaScript code that interacts with the page's scroll behavior. Here's a step-by-step guide on how to scroll the page in various ways using Selenium and Java:

1. Scroll Down

                        
// Scroll down by 500 pixels
JavascriptExecutor js = (JavascriptExecutor) driver;
js.executeScript("window.scrollBy(0, 500);");

2. Scroll to a Specific Element

                        
WebElement element = driver.findElement(By.id("elementId")); // Replace with your element's locator
JavascriptExecutor js = (JavascriptExecutor) driver;
js.executeScript("arguments[0].scrollIntoView(true);", element);

3. Scroll to the Bottom of the Page

                        
JavascriptExecutor js = (JavascriptExecutor) driver;
js.executeScript("window.scrollTo(0, document.body.scrollHeight);");

4. Scroll to the Top of the Page

                        
JavascriptExecutor js = (JavascriptExecutor) driver;
js.executeScript("window.scrollTo(0, 0);");
